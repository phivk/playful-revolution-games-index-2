  <files>src/components/InstagramFeed.tsx, src/components/NewsletterSignup.tsx</files>
  <action>
    **A. Create `src/components/InstagramFeed.tsx`:**

    - Add `'use client'` directive at top
    - Import `useState`, `useEffect` from 'react'
    - Import `Image` from 'next/image'
    - Define `InstagramPost` interface with fields: `id: string`, `media_url: string`, `thumbnail_url?: string`, `permalink: string`, `media_type: 'IMAGE' | 'CAROUSEL_ALBUM'`
    - Default export function `InstagramFeed()`
    - State: `const [posts, setPosts] = useState<InstagramPost[]>([])` and `const [loaded, setLoaded] = useState(false)`
    - useEffect with empty deps `[]`:
      - Calls `fetch('/api/instagram/feed')`
      - `.then(r => r.json()).then(data => { setPosts(data.posts ?? []); setLoaded(true); })`
      - `.catch(() => setLoaded(true))` — on error, set loaded=true so conditional render fires
    - Conditional render: `if (!loaded) return null` — no skeleton, no placeholder (per locked decision)
    - Conditional render: `if (loaded && posts.length === 0) return null` — section hidden entirely
    - The component renders its OWN section wrapper (including the border divider and heading) so that returning null hides everything — no section border or heading lingers in the DOM:
      ```tsx
      return (
        <section className="border-t border-gray-200 pt-8 mt-8">
          <h2 className="text-2xl font-bold text-gray-900 mb-6">@playrevolution</h2>
          <div className="grid grid-cols-3 gap-2">
            {posts.slice(0, 9).map((post) => (
              <a
                key={post.id}
                href={post.permalink}
                target="_blank"
                rel="noopener noreferrer"
                className="aspect-square overflow-hidden block"
              >
                <Image
                  src={post.media_url}
                  alt="Instagram post"
                  width={300}
                  height={300}
                  className="object-cover w-full h-full hover:opacity-80 transition-opacity"
                  unoptimized
                />
              </a>
            ))}
          </div>
        </section>
      );
      ```
    - Use `unoptimized` prop on Image (belt-and-suspenders against CDN domain issues; remotePatterns is also configured in next.config.ts)

    **B. Create `src/components/NewsletterSignup.tsx`:**

    - Add `'use client'` directive at top
    - Import `Script` from 'next/script'
    - Default export function `NewsletterSignup()`
    - Read the form ID from env: `const formId = process.env.NEXT_PUBLIC_SENDER_FORM_ID`
    - Return JSX:
      ```tsx
      // formId comes from NEXT_PUBLIC_SENDER_FORM_ID env var — set in .env.local and Vercel env vars
      <div>
        <Script
          src="https://cdn.sender.net/accounts_resources/universal.js"
          strategy="afterInteractive"
          onReady={() => {
            if (typeof window !== 'undefined' && (window as Window & { senderForms?: { render: () => void } }).senderForms) {
              (window as Window & { senderForms?: { render: () => void } }).senderForms?.render();
            }
          }}
        />
        <div
          className="sender-form-field"
          data-sender-form-id={formId}
        />
      </div>
      ```
    - Do NOT use `onLoad` — use `onReady` (fires on every mount including SPA re-navigation; `onLoad` fires only once)
    - The outer wrapper div needs no extra className — brand styling comes from Sender.net and the section wrapper in about/page.tsx
