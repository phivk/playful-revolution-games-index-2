# Requirements: Playful Revolution Games Index

**Defined:** 2026-02-20
**Core Value:** Facilitators can quickly find and view game instructions on mobile devices during sessions.

## v1 Requirements

### Core (Game Display)

- [ ] **CORE-01**: Display game catalog with ~30 games
- [ ] **CORE-02**: Game detail view shows title, description, tags, pillars, energy level
- [ ] **CORE-03**: Game detail view shows facilitation instructions (materials, setup, how to play)
- [ ] **CORE-04**: Mobile-friendly layout for viewing on phones

### Search & Filter

- [ ] **FILT-01**: Filter games by tags (Social Spontaneity, Group Circle Games, Collaborative, Competitive, Ball Games, Theatre Sports, Movement, Table Games)
- [ ] **FILT-02**: Filter games by pillars (Intellectual, Social, Physical)
- [ ] **FILT-03**: Filter games by energy level (Low, Medium, High)
- [ ] **FILT-04**: Search games by name or keyword
- [ ] **FILT-05**: Random game picker ("surprise me" feature)

### Design & UX

- [ ] **DESN-01**: Brand colors applied (Revolution Red #E53935, Play Green #43A047, Joy Yellow #FDD835, Deep Blue #1E3A8A) in bold color blocks
- [ ] **DESN-02**: Typography: Bold condensed sans-serif headlines (Bebas Neue/Anton), readable body font (Inter/Source Sans)
- [ ] **DESN-03**: "Street poster" aesthetic — bold, playful, slightly rough, zine-like
- [ ] **DESN-04**: Big bold buttons with strong color contrast
- [ ] **DESN-05**: Snappy, responsive interactions (buttons that jump, not fade)
- [ ] **DESN-06**: High contrast, accessible design, large tap targets
- [ ] **DESN-07**: Paper-like off-white background (#FAFAF7), not pure white

### CMS Integration

- [ ] **CMS-01**: Decap CMS admin interface for adding games
- [ ] **CMS-02**: Decap CMS admin interface for editing games
- [ ] **CMS-03**: Decap CMS admin interface for deleting games

### Deployment

- [ ] **DEPL-01**: Static site build/export
- [ ] **DEPL-02**: Deploy to Vercel or GH Pages

## v2 Requirements

- **V2-01**: User accounts for saving favorite games
- **V2-02**: Game ratings/reviews
- **V2-03**: User-submitted games (moderated)

## Out of Scope

| Feature | Reason |
|---------|--------|
| User accounts | Games are publicly accessible, no auth needed for v1 |
| User-submitted games | Admin-only for now to ensure quality |
| Interactive game tools (timers, scoring) | Focus on finding/viewing instructions first |
| Elegant slow animations | Keep it snappy and bold per brand |
